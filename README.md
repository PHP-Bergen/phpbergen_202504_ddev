### VS Code

[DDEV Manager](https://marketplace.visualstudio.com/items?itemName=biati.ddev-manager)

Install via VS Code quick open:
Hit `Ctrl+P` (ctrl+shift+p), paste the following and pres `enter`:
```
ext install biati.ddev-manager
```
